# next-transition-bar Example

This example demonstrates the features of `next-transition-bar`:

- **Basic navigation** between pages
- **Slow page** with simulated server delay
- **Middleware redirect** (`/old-path` → `/redirect-target`)
- **Programmatic navigation** using `router.push()`, `router.replace()`, `router.back()`

## Running locally

```bash
cd example
yarn install
yarn dev
```

Open [http://localhost:3000](http://localhost:3000).
